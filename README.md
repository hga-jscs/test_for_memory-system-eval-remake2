# test_for_memory-system-eval-remake2

本仓库当前维护的记忆后端聚焦为：

- MemGPT（Letta）
- RAPTOR
- LightRAG
- HippoRAG

一条命令顺序跑完四后端全量 benchmark：`python run_full_benchmark_all_backends.py`。

按“后端分开做”全量 benchmark：`python run_full_benchmark_split_backends.py`。

运行与排障请看：`docs/backend_runtime_guide.md`。
傻瓜式全量说明请看：`docs/full_benchmark_one_command.md`。
Anaconda 分开测评说明请看：`doc/anaconda_四后端分开全Benchmark测评指南.md`。
