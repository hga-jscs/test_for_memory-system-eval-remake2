## File Update Examples

### Adding to a table (reliable pattern using a unique marker)
```python
# Instead of trying to find the exact last row, use a marker like "<!-- TV_REVIEWS_TABLE_END -->"
# Insert the marker when creating the table for the first time.
# Then you can replace the marker with new rows + marker.

# First creation (if table doesn't exist):
table_header = "| Show/Creator | Genre | Sentiment | Feedback |\n"
table_marker = "<!-- TV_REVIEWS_TABLE_END -->"
initial_table = table_header + table_marker
create_file("user.md", initial_table)   # Or append to existing file

# Later, to add a new row:
old_marker = "<!-- TV_REVIEWS_TABLE_END -->"
new_row = "| A Great Show | Sci-Fi | Positive | \"character-driven with emotional depth\" |\n"
new_content = old_marker.replace(old_marker, new_row + old_marker)
result = update_file("user.md", old_marker, new_content)
```

### Updating statistics
```python
# Read the whole file, then replace a specific line using a unique identifier

new_line = "- sci_fi: positive=6, negative=0"

# If old_line not found exactly, update_file returns error message – handle it.
```

### Appending a new section (using a section marker)

## Memory Structure

- `entities/`: Information about people, places, organizations, shows, creators, networks, etc.
